Duplicate AJAX request in the browser

This page triggers a single ajax call by calling $.ajax('http://localhost:3000/some-route')

You can pop open your DevTools and you'll notice a single network request. However, after 121 seconds, you'll see in the server log, that a duplicate network request is handled by the server, perhaps issued by the browser?

If you issue a request via curl a la curl http://localhost:3000/some-route the request is not duplicated.
